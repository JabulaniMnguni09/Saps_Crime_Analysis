# South Africa Crime Statistics Analysis (2005–2016)

## Overview
Analysis of SAPS annual crime statistics across all nine South African 
provinces to identify which provinces carry the highest crime burden 
once population is accounted for, and which crime categories grew 
fastest over the eleven-year period.

## Business Question
- Which province carries the highest per capita crime burden?
- Which crime categories showed sustained growth between 2005 and 2016?

## Key Findings
- Western Cape recorded 8,422 crimes per 100,000 people — 66% higher 
than Gauteng and nearly double the national average
- Robbery at non-residential premises grew by 349% over the period
- Drug-related crime grew by 173%, escalating alongside robbery trends

## Tools Used
- Microsoft Excel — data joining, pivot tables, per capita calculations, 
trend analysis, and visualisation
